# JLS Recruit Site

Nuxt 3 + TypeScript + Tailwind CSS で作成した宅配ドライバー採用サイトです。

## ローカル確認

```bash
npm install
npm run dev
```

静的出力を確認する場合:

```bash
npm run build
```

## GitHub Pages 公開

1. GitHub に空のリポジトリを作成する
2. このプロジェクトをそのリポジトリに push する
3. GitHub の `Settings > Pages` で `Build and deployment` の `Source` を `GitHub Actions` にする
4. `main` ブランチに push すると、自動で公開されます

公開URLは通常、以下の形式になります。

```text
https://<github-user>.github.io/<repo-name>/
```
